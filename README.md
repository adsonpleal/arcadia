# arcadia

Arcadia is an open source CAD software written in Flutter. This is a personal project meant to improve my development skills.

## Project constraints

Since this project is meant to improve my skills, it has some self imposed constrains meant to improve specific knowledge areas:

 - Tests whenever possible
 - Everything should be docummented
 - No code generation
 - Heavy use of macros
 - No third-party dependencies outside dart/flutter packages
